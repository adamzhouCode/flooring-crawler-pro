# 🌍 Flooring Crawler Pro (智能地板拓客系统) - 极简使用说明

这是一个专为**地板行业**设计的高级自动化拓客与商业分析工具。它能根据您的定制需求，在全球范围内精准搜索地板行业的潜在客户（经销商、开发商、装修公司等），抓取其官网深层内容，并利用 AI 大模型自动完成评级、提炼联系方式和业务模式。

---

## 🚀 快速启动

在终端（Terminal）中运行以下命令启动系统界面：

```bash
streamlit run app.py
```
> 系统会自动在浏览器中打开 Web 操作界面 (通常为 `http://localhost:8501`)。

---

## 🛠️ 操作三步走

### 1. 设定您的“捕猎区” (左侧边栏)
* **搜索区域/语言**：选择您想开发的国家市场。
  * 内置了 **中国**、**美国**、**南非**、**澳洲** 等多个主流区域的专用配置。
  * 如果想开发小众市场（如沙特、墨西哥等），请选择 **“🌐 自定义国家/语言 (Custom)”**，并输入对应的两位数国家代码（如 `ae`, `mx`）和语言代码。
* **目标行业**：选择您想要合作的客户类型（经销商、零售门店、开发商、室内设计等）。系统会自动加载不同身份的 AI 评估标准。
* **目标城市**：输入您想深耕的具体城市（如：`上海`, `Dallas`, `Cape Town`）。

### 2. 微调与执行 (右侧操作区)
* **搜索指令**：系统会根据前面的选择，**自动生成**一段极其精准的高级搜索语法（不仅包含了精准的行业关键词，还自动屏蔽了各种 B2B黄页、招聘网站、骗局加盟站等污染源）。*一般情况下直接保持默认即可。*
* **点击【🚀 开始自动化拓客任务】**，坐和放宽。这期间爬虫引擎和 AI 会在云端为您并发执行阅读、提取、过滤和分析任务。

### 3. 查看报表与导出 (底部结果区)
* 任务结束后，界面会生成一个包含**潜力评分**的高颜值数据面板。
* **导出 Excel**：点击面板下方的 `📥 导出 Excel` 按钮。系统会保留按“潜力核心度”降序的排列格式，直接生成易于跟进审阅的最终商业线索表格。所有即使是外文网站的复杂判定和业务介绍，AI 也已为您全部贴心地翻译成了**中文总结**！

---

## 🔑 必要前提与注意事项

1. **API Keys 配置**: 确保根目录的 `.env` 文件已正确配置以下账密，否则系统无法启动引擎：
   - AI 分析核心 (任选其一): `DEEPSEEK_API_KEY` 或 `OPENAI_API_KEY`
   - 搜索雷达 (任选其一): `SERPER_API_KEY`, 或 `GOOGLE_API_KEY` + `GOOGLE_CX`
2. **纯粹原则**: 爬虫层经过极其严格的把控，在预检阶段会自动砍掉没有商业开发价值的网站。因此产生了一定的**自然淘汰率**（如直接屏蔽亚马逊、Yelp 等）。这是正常的过滤器保护机制。
3. **网络连通**: 如果使用 Google / OpenAI 等需要访问国际大网的服务，请确保您的终端具备稳定的国际网络环境。
